Introduction
  • HTML stands for Hyper Text Markup Language

    • HTML is the standard markup language for creating Web pages and the most basic building block of the Web. It defines the meaning and structure of web content.

    "Hypertext" refers to links that connect web pages to one another, either within a single website or between websites. Links are a fundamental aspect of the Web. By uploading content to the Internet and linking it to pages created by other people, you become an active participant in the World Wide Web.

    • Tags in HTML
    HTML uses tages to define type of content in that elements.
  • The opening tag:

    This consists of the name of the element (in this example, p for paragraph), wrapped in opening and closing angle brackets. This opening tag marks where the element begins or starts to take effect. If you wants to give some characterstics to your element like color and size.

  • The content:

    This is where main content of the element is written i.e. in content are generally paragraph,topics, etc. are written. Contents can have child elements like p, h1, etc.

  • The closing tag:

    This is the same as the opening tag, except that it includes a forward slash before the element name. This marks where the element ends.

    • Some fundamental HTML tags are:

    1. <!DOCTYPE html >   : Is defines that the document is HTML5 documentation
    2. <HTML>   : element is the root element of an HTML page
    3. <head>   : Contains meta information of pages.
    4. <title>   : Specifies title for HTML page. Which shows in page's tab
    5. <body>   : It defines document's body.
    6. <h1>   : Defines a large heading.
    7. <p>   : element defines a paragraph.
    8. <section>   : Defines new Section.
